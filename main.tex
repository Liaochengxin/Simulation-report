\begin{abstract}
\sloppy
本项目基于 Geant4 构建了一个多尺度肿瘤放疗模拟程序，比较常规照射（gamma、proton）与硼中子俘获治疗（BNCT）的剂量沉积特征。Q1 在简化人体结构中对比 gamma 与 proton 的深度剂量曲线、二维能量沉积分布、LET 谱及能量扫描结果；Q2 在肿瘤区内构建正常/癌症细胞的混合结构，研究 $^{10}\mathrm{B}$ 均匀分布与壳层分布对整细胞剂量、细胞核剂量及 BNCT 二次粒子产额的影响。

结果表明：gamma 能量沉积均匀，并未见显著能量沉积区域，proton 可通过能量调节使 Bragg peak 落在肿瘤深度附近；在理想化混合细胞模型中，含 $^{10}\mathrm{B}$ 肿瘤细胞获得的剂量明显高于同场不含 $^{10}\mathrm{B}$ 的对照细胞；$^{10}\mathrm{B}$ 浓度主要影响 BNCT 反应活性与靶细胞剂量，相对中子注量主要放大累计剂量，对剂量局域化比例无显著影响。
\par\end{abstract}

\section{基本研究内容}

本项目围绕肿瘤放射治疗中不同粒子、不同治疗机制的剂量选择性展开，核心内容包括：

\begin{enumerate}
    \item 构建简化人体（头部、颈部、躯干、双腿），在躯干内设置肿瘤区。
    \item 比较 gamma 与 proton 照射下的肿瘤区与正常细胞区域剂量沉积对比。
    \item 通过能量扫描研究入射能量对剂量空间分布的影响。
    \item 构建肿瘤区的正常/肿瘤细胞结构，开展 BNCT 细胞尺度模拟。
    \item 比较 $^{10}\mathrm{B}$ 均匀分布与壳层分布对整细胞剂量、细胞核剂量及 $\alpha$+$^{7}\mathrm{Li}$ 产额的影响。
    \item 扫描 $^{10}\mathrm{B}$ 浓度和相对中子注量，分析两者对 BNCT 治疗效果的影响。
\end{enumerate}

物理列表（Physics List）采用 \verb|QGSP_BIC_HP|，包含电磁过程、质子强子过程和高精度低能中子过程，可在同一程序中处理 Q1 的 gamma/proton 照射和 Q2 的热中子 BNCT。

$^{10}\mathrm{B}$ 同位素参数为 \verb|Z = 5|、\verb|A = 10|、摩尔质量 \verb|10.012937 g/mol|。硼含量用按质量计的 ppm 表示，其中 $1\,\mathrm{ppm}=10^{-6}$ 的质量分数：

\begin{equation}
    \omega_{\mathrm{B}} = C_{\mathrm{B}}(\mathrm{ppm}) \times 10^{-6}.
\end{equation}

Q2 中使用的 $C_{\mathrm{B}}=5.0\times 10^{5}\,\mathrm{ppm}$ 远高于正常治疗的质量分数，对应 $\omega_{\mathrm{B}}=0.5$，即 50\% 质量分数，主要用于在有限事件数下增强反应统计量。

\subsection{辐射剂量说明}

吸收剂量定义为：

\begin{equation}
    D = \frac{E_{\mathrm{dep}}}{m}.
    \label{eq:dose}
\end{equation}

其中，$D$ 为吸收剂量，表示单位质量物质吸收的电离辐射能量；$E_{\mathrm{dep}}$ 为沉积能量，即射线或粒子在特定几何区域内由于相互作用而损失并留在该区域内的总能量；$m$ 为接收该段能量的目标区域质量。

各监测指标的直观含义与计算方法如下：

\begin{itemize}
    \item \textbf{肿瘤/正常细胞平均剂量}：每个入射粒子在目标区域内沉积的平均剂量，正常细胞质量由人体扣除肿瘤细胞后计算得到。
    \item \textbf{Q1 深度剂量与二维热图}：展示剂量在空间（一维深度、二维平面）上的相对分布形状，数据已做归一化处理。
    \item \textbf{Q1 线能量转移}：按每个 step 的 \verb|E_dep / stepLength| 统计并对 step 数归一化，展示能量的平均损失速率。
    \item \textbf{Q2 整细胞剂量}：整次运行中沉积在单个球形细胞（含细胞核）内的总能量除以细胞质量，对全部 2048 个细胞取平均，未发生沉积的细胞以零计入。
    \item \textbf{Q2 细胞核剂量}：沉积在单个细胞核内的总能量除以细胞核质量，平均方式同上。
    \item \textbf{Q2 细胞剂量沉积}：同一 \verb|(x,z)| 内各 y 层整细胞剂量之和，用于展示空间分布趋势。
    \item \textbf{$\alpha$+$^{7}\mathrm{Li}$ 产额}：运行中生成的 $\alpha$ 与 $^{7}\mathrm{Li}$ 二次粒子数之和，作为 BNCT 反应活性指标。
\end{itemize}

注：Q1 使用每事件宏观区域计分，Q2 使用整次运行累计的单细胞计分，两者质量尺度不同，不能直接比较绝对剂量。

\subsection{Q1 小结}

gamma 无 Bragg peak，剂量相对平均；proton 可通过能量调节将 Bragg peak 放置于肿瘤深度，45 MeV 与当前几何匹配较好。gamma 能量扫描主要改变穿透深度，proton 能量扫描直接改变 Bragg peak 位置。质子治疗相对 gamma 的核心优势在于剂量空间可控性。

\section{Q2：BNCT 细胞尺度模拟}

\subsection{研究目标}

BNCT 通过含硼载体药物使肿瘤细胞选择性富集 $^{10}\mathrm{B}$，再用低能中子照射，引发：

\begin{equation}
    ^{10}\mathrm{B} + n \rightarrow \alpha + ^{7}\mathrm{Li}.
\end{equation}

反应产物 $\alpha$ 和 $^{7}\mathrm{Li}$ 射程仅约 $10\,\mu\mathrm{m}$，能量主要沉积在含硼细胞及其近邻区域，实现细胞尺度的剂量局域化。

Q2 的目标是：在同一中子场内，比较 $^{10}\mathrm{B}$ 均匀分布与壳层分布两种情形下，含硼肿瘤细胞与不含硼对照细胞的整细胞剂量、细胞核剂量和 $\alpha$+$^{7}\mathrm{Li}$ 产额差异，并通过 $^{10}\mathrm{B}$ 浓度和中子注量扫描分别分析两个参数的作用。

\subsection{\texorpdfstring{$^{10}\mathrm{B}$}{10B} 浓度扫描}

固定中子能量，扫描 $^{10}\mathrm{B}$ 浓度，每点 200000 个入射粒子：

\begin{center}
    \verb|1000, 3000, 10000, 30000, 100000, 300000, 500000 ppm|
\end{center}

定义剂量局域化指标：

\begin{equation}
    S_{\mathrm{cell}} =
    \frac{D_{\mathrm{tumor,cell}}}{D_{\mathrm{tumor,cell}} + D_{\mathrm{normal,cell}}},
\end{equation}

\begin{equation}
    S_{\mathrm{nucleus}} =
    \frac{D_{\mathrm{tumor,nucleus}}}{D_{\mathrm{tumor,nucleus}} + D_{\mathrm{normal,nucleus}}}.
\end{equation}

$S$ 越接近 1，剂量越集中于含 $^{10}\mathrm{B}$ 细胞。

图\ref{fig:q2-b10-scan}展示浓度扫描的四个子图：剂量局域化比例、累计细胞核剂量、$\alpha$+$^{7}\mathrm{Li}$ 产额、肿瘤核剂量与整细胞剂量之比。

\begin{figure}[H]
    \centering

随 $^{10}\mathrm{B}$ 浓度从 1000 ppm 升至 500000 ppm，uniform 的 $\alpha$+$^{7}\mathrm{Li}$ 产额由 45 增至 1432，shell 由 41 增至 875，两条曲线均呈单调上升趋势，证实了“提高 $^{10}\mathrm{B}$ 浓度可增加 BNCT 反应活性”的预期。uniform 模式的肿瘤核剂量总体高于 shell；低浓度区因俘获反应数较少，核剂量和局域化比例存在残余统计涨落。

\subsection{Q2 小结}

在本次研究中，通过构建混合细胞模型，验证了 BNCT 的核心物理特征：BNCT 的细胞靶向选择性依赖于 $^{10}\mathrm{B}$ 在肿瘤细胞中的富集。硼元素在细胞内均匀分布（uniform）时，粒子能量更易直达细胞核，对肿瘤细胞核的剂量沉积效果显著优于外壳分布（shell）。中子注量主要决定累计剂量大小，但不会改变由 $^{10}\mathrm{B}$ 富集与分布决定的细胞选择性。

\section{结论}

本项目完成了从宏观人体能量沉积到微观细胞 BNCT 计分的 Geant4 多尺度模拟。

Q1 表明：gamma 能量沉积弥散，无明显峰值；proton 可通过能量调节将 Bragg peak 置于肿瘤深度，45 MeV 与当前几何匹配最佳，质子治疗的关键优化变量是入射能量。

Q2 表明：在 $^{10}\mathrm{B}$ 仅富集于肿瘤细胞的理想模型中，$\alpha$/$^{7}\mathrm{Li}$ 短射程产物形成明显的细胞尺度剂量局域化；$^{10}\mathrm{B}$ 浓度影响反应活性与靶细胞剂量，uniform 分布更有利于提高肿瘤核剂量；相对中子注量主要控制累计剂量和二次粒子产额，不改变由 $^{10}\mathrm{B}$ 分布决定的细胞选择性。

因此，评价 BNCT 需同时考察 $^{10}\mathrm{B}$ 摄取分布、细胞剂量局域化、二次粒子产额和细胞核剂量，而不能仅凭绝对剂量判断。本项目为进一步引入真实组织材料、实际硼药物摄取分布、重复统计、RBE 修正和细胞生存模型提供了建模基础。

\section{局限性}

虽然模型成功反映了物理趋势，但由于以下简化，结果不能直接等同于临床真实情况：

\begin{itemize}
    \item 整个三维人体模型全部用纯水代替，忽视了骨骼、肺部和脂肪等真实组织对中子的散射与衰减。同时，Q1 宏观剂量由于模拟粒子数有限，导致 Bragg 峰的曲线形状较为粗糙。
    \item 材料与浓度设定偏离临床。为了在 Q2 微观模拟中获得明显的增强信号，采用了极高的硼浓度（500000 ppm），且假设正常对照细胞完全不吸附硼。现实中正常组织也会误吸收少量硼药物，因此当前模型明显高估了细胞的选择性。
    \item 缺乏能量与生物效应归一化。当前仅对比了纯物理剂量，未引入 RBE（相对生物效应）修正、DNA 损伤模型或细胞存活曲线。此外，由于不同粒子对照时未统一入射总能量或处方剂量，目前的结论仅限于定性比较物理剂量的局域化特征，无法直接外推为实际的细胞杀伤率。
\end{itemize}
